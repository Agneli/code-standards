Otimização para Motores de Busca

Uma parte essencial de um bom web design e desenvolvimento é o SEO (Search Engine Optimization). Um bom estruturado código é a chave para assegurar que uma web page não é só corretamente indexada por motores de busca, mas torná-la acessível a pessoas com capacidades limitadas na web.

  • Imprimir práticas recomendadas para CSS.
  • Site/app deve ser ajustado de acordo com as Diretrizes da Resolução do Navegador.
  • Site/app deve ser compatível com os requisitos descritos em Teste em Navegadores e Suporte.
  • Esteja ciente das melhores práticas de Acessibilidade, tal como o 508 e os padrões WCAG:

Indexabilidade

Nós devemo usar marcação semântica porque é legível e lógica quando o JavaScript e o CSS estão desligados. Todo conteúdo de uma página em HTML; nós não queremos usar iframes ou JavaScript para carregar conteúdo indexável.

Este permite que a página indexe corretamente pelos motores de busca e também permite que os usuários abram-os em novas abas e janelas.

Otimização

A tag title devem possuir palavras-chave para uma página única. Os títulos devem ser únicos para cada página. Cabeçalhos (h1, h2 etc) devem formar um contorno do documento e representar as mais importantes palavras-chave para a página. URLs devem ser legível com palavras-chave alvo primárias presentes neles:

http://domain.com/mens-shoes/basketball/jordan/jordan-mens-ajf-6-basketball-shoe/

vs

http:// domain.com/ecomm.cfm?view=prod&prodId=23425

Google's SEO Report Card

Google's SEO Report Card é um esforço para fornecer às equipes de produtos do Google com ideias sobre como eles podem melhorar a páginas de seus produtos usando otimizações simples e aceitáveis. Estas otimizações são destinadas para não apenas ajudar nos motores de busca entendam o conteúdo da sua página melhor, mas também melhoraria a experiência desses usuários.

Quando visitamos estes sites medidas simples, tais como fixação de ligações 404s e links quebrados, simplificando a escolha URL, e proporcionando mais fáceis de entender títulos e trechos de suas páginas podem beneficiar os usuários e os motores de busca.
